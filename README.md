## Features

- App Router is used
- Mock data is returned by the Route Handler

Here are the steps to run the project locally.

1. Clone the repository

   ```bash
   git clone https://github.com/GentaAmeku/x-clone.git
   ```

1. Install dependencies

   ```bash
   pnpm install
   ```

1. Run the project

    ```bash
    npm run dev
    ```
